# @amlplugins/google-cloud-apigee

Native `@google-cloud/apigee-registry` SDK re-exported under the `@amlplugins` namespace.

> Apigee Registry.

## Install

```bash
npm install @amlplugins/google-cloud-apigee
```

## Use

```ts
import { /* named exports from @google-cloud/apigee-registry */ } from "@amlplugins/google-cloud-apigee";
// or
import sdk from "@amlplugins/google-cloud-apigee";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
